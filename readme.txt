--Instrucciones/Shortcuts--
P -> herramienta para seleccionar
C -> Desplegar círculo
B -> Desplegar curva de bezier
Nota: Después de haber dado los respectivos clics la curva aparecerá en pantalla dando a clic derecho
Q -> Desplegar rectángulo
T-> Desplegar triángulo 
Nota: En este caso son dos clics y después se puede arrastrar para el tercer punto.
E -> Desplegar elipse.
figura seleccionada + X -> eliminar
figura seleccionada la figura
figura seleccionada + U -> subir todos los niveles la figura
figura seleccionada + D -> bajar UN nivel la figura
figura seleccionada + M -> Subir UN nivel de la figura
figura seleccionada + K -> Trasladar figura haciendo clic y arrastrando
